Modelos Candidatos

En esta sección se presentan los modelos entrenados para predecir las ventas globales de videojuegos. La comparación se realiza utilizando RMSE, MAE y R² sobre el conjunto de prueba.

Variable objetivo log_Global_Sales
Tipo de problema Regresión
Modelo seleccionado Gradient Boosting
Modelo A

Regresión Lineal

Modelo base utilizado como punto de comparación. Asume una relación lineal entre las variables explicativas y las ventas globales del videojuego.

Rol dentro del análisis

Sirve como baseline para evaluar si modelos más complejos logran mejorar la capacidad predictiva. Su principal limitación es que no captura bien relaciones no lineales entre plataforma, género, editora y ventas.

RMSE 0.3582
MAE 0.2307
R² 0.2226

Validación cruzada R²: 0.2493 ± 0.0176

Modelo B

Random Forest

Modelo basado en un conjunto de árboles de decisión. Permite capturar relaciones no lineales y combinaciones complejas entre las variables.

Rol dentro del análisis

Mejora considerablemente frente a la Regresión Lineal, ya que puede modelar interacciones entre variables como plataforma, género, editora, puntuaciones y cantidad de reseñas.

RMSE 0.2930
MAE 0.1766
R² 0.4799

Validación cruzada R²: 0.4827 ± 0.0072

Modelo C

Gradient Boosting

Modelo de boosting basado en árboles de decisión. Construye modelos de forma secuencial, donde cada nuevo árbol intenta corregir los errores cometidos por los árboles anteriores.

Rol dentro del análisis

Fue el modelo con mejor desempeño general dentro de la versión actualizada. Obtuvo el mayor R² en el conjunto de prueba, por lo que fue seleccionado como modelo final para la aplicación web.

RMSE 0.2820
MAE 0.1709
R² 0.5180

Modelo final utilizado en la predicción web

Comparación general

Modelo RMSE MAE R² Conclusión
Regresión Lineal 0.3582 0.2307 0.2226 Modelo base, pero con bajo poder predictivo.
Random Forest 0.2930 0.1766 0.4799 Mejora relevante al capturar relaciones no lineales.
Gradient Boosting 0.2820 0.1709 0.5180 Mejor desempeño en la versión actualizada, seleccionado como modelo final.

Modelo Final Seleccionado: Gradient Boosting

El modelo seleccionado para la aplicación fue Gradient Boosting, ya que obtuvo el mejor desempeño dentro de la comparación realizada en la versión actualizada del proyecto. En particular, presentó el mayor R² en el conjunto de prueba y un menor error respecto a los modelos candidatos.

Este modelo es adecuado para el problema porque permite capturar relaciones no lineales entre las variables del mercado de videojuegos, como la plataforma, el género, la editora, el año de lanzamiento y las puntuaciones de crítica y usuarios. Además, al trabajar de forma secuencial, mejora progresivamente la predicción corrigiendo errores de modelos anteriores.
