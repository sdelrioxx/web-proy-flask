Modelos Candidatos

En esta sección se presentan los modelos entrenados para predecir las ventas globales de videojuegos. La comparación se realiza utilizando RMSE, MAE y R² sobre el conjunto de prueba.

Variable objetivo log_Global_Sales
Tipo de problema Regresión
Modelo seleccionado XGBoost
Modelo A

Regresión Lineal

Modelo base utilizado como punto de comparación. Asume una relación lineal entre las variables explicativas y las ventas globales del videojuego.

Rol dentro del análisis

Sirve como baseline para evaluar si modelos más complejos logran mejorar la capacidad predictiva. Su principal limitación es que no captura bien relaciones no lineales entre plataforma, género, editora y ventas.

RMSE 0.3582
MAE 0.2307
R² 0.2226

Validación cruzada R²: 0.2493 ± 0.0176

Modelo B

Random Forest

Modelo basado en un conjunto de árboles de decisión. Permite capturar relaciones no lineales y combinaciones complejas entre las variables.

Rol dentro del análisis

Mejora considerablemente frente a la Regresión Lineal, ya que puede modelar interacciones entre variables como plataforma, género, editora, puntuaciones y cantidad de reseñas.

RMSE 0.2930
MAE 0.1766
R² 0.4799

Validación cruzada R²: 0.4827 ± 0.0072

Modelo C

XGBoost

Modelo de gradient boosting basado en árboles. Construye modelos de forma secuencial, corrigiendo los errores de los árboles anteriores.

Rol dentro del análisis

Fue el modelo con mejor desempeño general, obteniendo el menor error y el mayor R² en el conjunto de prueba. Por esta razón, fue seleccionado como modelo final para la aplicación web.

RMSE 0.2800
MAE 0.1708
R² 0.5250

Validación cruzada R²: 0.5276 ± 0.0109

Comparación general

Modelo RMSE MAE R² Conclusión
Regresión Lineal 0.3582 0.2307 0.2226 Modelo base, pero con bajo poder predictivo.
Random Forest 0.2930 0.1766 0.4799 Mejora relevante al capturar relaciones no lineales.
XGBoost 0.2800 0.1708 0.5250 Mejor desempeño, seleccionado como modelo final.

Modelo Final Seleccionado: XGBoost

El modelo seleccionado para la aplicación fue XGBoost, ya que obtuvo el mejor desempeño en las métricas evaluadas. En comparación con los otros modelos, presentó el menor RMSE, el menor MAE y el mayor R², lo que indica una mejor capacidad para aproximar las ventas globales de videojuegos.

Además, XGBoost es adecuado para este problema porque puede capturar relaciones no lineales entre las variables del mercado de videojuegos, como la plataforma, el género, la editora, el año de lanzamiento y las puntuaciones de crítica y usuarios.
